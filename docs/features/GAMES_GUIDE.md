# 🎮 Game Commands Guide

All game commands are now fully integrated and ready to use!

## Available Games

### 1. Magic 8-Ball (.8ball)

Ask the magic 8-ball any yes/no question.

```
.8ball Will I win the lottery?
```

### 2. Dice Roll (.dice)

Roll two dice and see the total.

```
.dice
```

### 3. Coin Flip (.coinflip or .coin)

Flip a coin - heads or tails.

```
.coinflip
```

### 4. Rock Paper Scissors (.rps)

Play rock paper scissors against the bot.

```
.rps rock
.rps paper
.rps scissors
```

### 5. Number Guessing (.guess)

Guess a number between 1-100. Bot gives hints.

```
.guess 50
```

Start a new game with `.guess` (no number).

### 6. Trivia (.trivia)

Answer random trivia questions from various categories.

```
.trivia
```

Reply with your answer (a, b, c, or d).

### 7. Slot Machine (.slot)

Spin the slot machine and try to match symbols.

```
.slot
```

Match 3 symbols to win!

### 8. Math Challenge (.math)

Solve random math problems (addition, subtraction, multiplication).

```
.math
```

Reply with your answer.

## How to Access

1. Type `.menu` to see all commands (English)
2. Type `.menuita` to see all commands (Italian)
3. Games section is now included in both menus

## Features

- All games work in both private chats and groups
- No special permissions required
- Instant responses
- Fun and interactive gameplay
- Session-based games (guess, trivia, math) track your progress

## Testing

All game commands have been tested and are working correctly. The bot will:

- Respond immediately to game commands
- Track game state for multi-turn games
- Provide clear instructions and feedback
- Handle invalid inputs gracefully

Enjoy the games! 🎮
