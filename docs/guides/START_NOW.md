## Troubleshooting

### Bot not responding?

1. Check WhatsApp Web is closed
2. Check terminal shows "Bot connected successfully"
3. Make sure you're using `.` prefix (`.ping` not `ping`)

### WhatsApp Web frozen?

1. Close WhatsApp Web
2. Stop bot: `Ctrl+C`
3. Wait 10 seconds
4. Start bot: `npm start`

### Need fresh start?

```bash
rm -rf auth_info/
npm start
# Scan QR again
```
