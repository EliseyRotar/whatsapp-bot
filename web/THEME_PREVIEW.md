## 🎨 Accent Usage

Red accents are used for:

- ✅ Primary buttons
- ✅ Active navigation items
- ✅ Status indicators (online)
- ✅ Links and interactive elements
- ✅ Progress bars
- ✅ Toggle switches (ON state)
- ✅ Focus states
- ✅ Important badges

## 🌈 Semantic Colors

```
Success: #22C55E (Green)  - Confirmations, success states
Warning: #F59E0B (Orange) - Warnings, cautions
Error:   #EF4444 (Red)    - Errors, critical alerts
Info:    #3B82F6 (Blue)   - Information, tips
```

## 💫 Animations

All animations maintain 60fps:

- Button hover: Scale + shadow
- Card hover: Lift effect
- Toggle switch: Smooth slide
- Page transitions: Fade in
- Toast notifications: Slide in

## 🔍 Accessibility

- **Contrast Ratio**: 4.5:1+ for all text
- **Focus Indicators**: 2px red outline
- **Touch Targets**: 44px minimum
- **Keyboard Nav**: Full support
- **Screen Readers**: ARIA labels
